Este juego se llama Spaceship.
Es una nave la cual tiene que destruir a las naves enemigas. 

INSTRUCCIONES:

-Para mover la nave usa las teclas dirección.
-Para disparar usa la tecla "e".
-Por cada nave enemiga destruida se te dara 100 puntos.
-Puedes pausar el juego con la barra espaciadora.

DIFICULTADES:

El posicionamiento de las insatancias en la consola.
Hacer los disparos de la nave principal y enemiga.
Las colisiones entre bala y nave ; nave y enemigo.
